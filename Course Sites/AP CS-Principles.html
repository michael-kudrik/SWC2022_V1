Students will utilize resources such as MIT App Inventor, JavaScript and others to display their understanding of computer science principles and create their own computational artifacts. AP Computer Science Principles is an essential High School coding class. It provides students with challenging projects that test critical skills integral in real world. It also maintains a balance beween projects, tests and hands on activities to keep the students engaged and interested. Most of the course focuses on the core algorithms used in today's fields as well as focusing on code that implements it. Java is the main focus language as it is used in most modern applications, devices etc. All in all, this class provides tools for a successful future in the Tech field.
